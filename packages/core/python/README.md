# caracalai-core

Core utilities shared across Caracal Python packages: scope evaluation and primitives.

Part of [Caracal](https://github.com/Garudex-Labs/caracal) — a security-first authority and delegation system for AI agents.

## Install

```bash
pip install caracalai-core
```
