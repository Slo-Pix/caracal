## Contribution boundaries

- Top-level `caracal` stays limited to runtime lifecycle and `caracal run`.
- Product management belongs to Console, Admin SDK, or Control API.
- Security reports must follow [Disclosure Policy](/security/disclosure/), not public issues.
- Open-source code must not import, reference, or depend on enterprise-only code.
