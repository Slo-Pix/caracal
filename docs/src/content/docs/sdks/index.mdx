## Runtime requirements

| Ecosystem | Current package target |
| --- | --- |
| Node.js | Node `>=22` for published TypeScript packages that declare an engine. |
| Python | Python `>=3.12` for published Python packages. |
| Go | Module paths under `github.com/garudex-labs/caracal/packages/...`; current modules declare Go `1.26` where specified. |

## How to choose

| Need | Start with |
| --- | --- |
| Spawn agents, delegate, inject headers, or route through Gateway | [TypeScript SDK](/sdks/typescript/), [Python SDK](/sdks/python/), or [Go SDK](/sdks/go/) |
| Verify mandates inside a resource server | [Identity Package](/sdks/identity/) or [MCP Transport](/sdks/transport-mcp/) |
| Exchange tokens directly against STS | [OAuth Package](/sdks/oauth/) |
| Track revoked sessions and delegation edges | [Revocation Package](/sdks/revocation/) and [Redis Token State](/sdks/connectors/redis/) |
| Automate zones, resources, policies, audit, agents, or delegation | [Admin Package](/sdks/admin/) |
| Protect framework-specific servers | [Connectors](/sdks/connectors/) |
