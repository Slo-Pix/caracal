Automate setup

Use the bundled scripts to create Lynx providers, resources, and the baseline policy from the current catalog.

1. Create the control key

Create the key inside the Lynx zone in Caracal Console. The key is already bound to that zone.

2. Copy the credentials

Copy the one-time client id and secret from the Control menu. No zone value is required.

3. Run provision

Run the script from this example directory. It reads the provider catalog and creates the Caracal objects.
