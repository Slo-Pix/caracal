Setup

Confirm your environment is ready to run the {{ company }} demo.

1 · Start the Caracal stack

Coordinator + Gateway from the monorepo's infra/docker compose. Lynx Capital uses these for identity, delegation, and resource-aware provider access.

# from repo root
docker compose -f infra/docker/docker-compose.yml up -d

2 · Start the provider mocks

13 REST providers, 2 gRPC services, and 1 MCP server, behaving like real external APIs (auth, idempotency, webhooks, rate limits).

# from examples/lynxCapital
docker compose -f _mock/docker-compose.yml up -d --build

3 · Configure environment

Copy the template and fill in OPENAI_API_KEY. Caracal vars already point at the local stack.

cp .env.example .env
# edit .env → set OPENAI_API_KEY=sk-...

4 · Run Lynx Capital

uv run uvicorn app.main:app --reload --port 8000

5 · Validate

Confirms OPENAI_API_KEY is set, all four CARACAL_* env vars are present, and the coordinator + gateway are reachable.
