# Product Discovery

Use to understand a user's product, workflows, resources, providers, auth model, and credential handling before Caracal SDK integration.

## Procedure

1. Read README, app entry points, routes, auth files, config files, and environment examples.
2. Identify product purpose, users, workflows, resources, providers, and integrations.
3. Identify current authentication, authorization, and resource access patterns.
4. Identify where secrets, API keys, and provider credentials are managed.
5. Summarize understanding and ask the user to confirm.

Do not propose implementation before confirmation.
