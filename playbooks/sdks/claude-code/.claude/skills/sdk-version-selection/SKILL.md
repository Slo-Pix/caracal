# SDK Version Selection

## Procedure

1. Detect package manager and dependency files.
2. Detect runtime and framework versions.
3. Check whether a Caracal SDK is already installed.
4. Verify the correct SDK version against Caracal and SDK documentation.
5. Use only official SDK terminology, methods, types, and configuration structures.

Never generate SDK code from memory when version-specific documentation is available.
