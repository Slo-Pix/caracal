# Integration Planning

## Procedure

1. Confirm product and architecture understanding.
2. Identify the user's desired Caracal outcome.
3. Identify recommended integration points.
4. Identify optional integration points.
5. Identify what should remain unchanged.
6. Separate Admin API setup from runtime application code.
7. Ask for user confirmation before implementation.

Use the `Product Understanding`, `Architecture Understanding`, `Proposed Integration`, and `User Confirmation` format from `CLAUDE.md`.
